# Recommendation-System-Decodelabs
Project Week 3
Recommendation System Using MovieLens Dataset

This project is a simple Recommendation Systems built using the popular MovieLens Dataset.
The system recommends movies to users based on their favorite genre using basic data filtering techniques in Python.

The main objective of this project is to understand the fundamentals of recommendation systems, dataset handling, and personalized movie suggestions.

Features
Load and process the MovieLens dataset
Take user input for preferred movie genre
Recommend movies based on genre matching
Display top movie recommendations
Beginner-friendly implementation using Python and Pandas
Technologies Used
Python
Pandas
CSV Dataset Handling
Dataset Used

This project uses the MovieLens dataset provided by GroupLens Research.

Dataset includes:

Movie Titles
Genres
Movie IDs
Project Workflow
Load the MovieLens dataset
Display dataset preview
Take user's favorite genre as input
Filter movies matching the selected genre
Display recommended movies
Recommendation Logic

The recommendation system works using a simple content-based filtering approach:

User enters a movie genre
The system searches movies containing that genre
Matching movies are recommended to the user
